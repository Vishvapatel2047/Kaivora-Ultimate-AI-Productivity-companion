# Kaivora

View app in AI Studio: https://ai.studio/apps/8ec8c067-fcb5-4732-9bac-59f4a6354fa6
An AI-powered productivity companion that helps users manage tasks, build habits, protect focus, and maintain a healthy work-life balance.

## Features
- AI Daily Coach
- Voice Task Input
- Rescue Mode
- Smart Notifications
- AI Assistant
- Goal Management
- Life Timeline
- Streaks & Streak Shields
- Achievement Badges
- Hall of Wins
- Focus Sessions
- Notification Diagnostics

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- Google Gemini API
- Node.js
- Express.js

npm install
npm run dev
